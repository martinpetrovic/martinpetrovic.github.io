Martin Luther King Junior games is a game development studio based in Zagreb, Croatia. We create games in Python using the Pygame module and have created Potapanje brodova. We are also planning to create other games but for the time being we do not have a project that we are working on however we are always open for ideas and new challenges.

Potapanje brodova is an offline two player game where you spread your ships across the sea and your enemy does the same. In the second phase of the game you are supposed to shoot and sink enemy ships and the winner is the person who sinks all enemy ships first. We have also created a profile and score system for the game so you can track all of your scores. We plan on updating the game to allow online multyplayer, fix some bugs and overall make the game look better.

The game is not very demanding hardware wise however there is a requirement software wise and that is that you have Windows 10 installed. Also the game will not work if your screen resolution je less than 1280 by 720 pixels or HD. The game can use up a lot of RAM however if you play normally you will not experience this problem.
